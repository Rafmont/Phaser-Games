Phaser JS é um framework para desenvolvimento web que disponibiliza para o desenvolvedor uma enorme quantidade de funções para a criação de seus jogos.

Além disso a comunidade de desenvolvimento do Phaser JS é rica em conteudo para o aprendizado, facilitando assim desenvolvedores iniciantes começar sua jornada no desenvolvimento de jogos! Para saber mais sobre a comunidade pode-se acessar a sessão de comunidade aqui.

Os exemplos usados nesse site estão com os respectivos códigos fontes disponíveis para o acesso, para que dessa forma sirva de exemplo para outros desenvolvedores que queiram criar aventuras utilizando o Phaser!
